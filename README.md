<!-- ABOUT THE PROJECT -->
## Objective

The primary goal of this Python program is to monitor a specific online marketplace for the availability of a predefined article and send email notifications to users as soon as the item becomes accessible for purchase.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## Key Features
* **Article Monitoring**: The program continuously checks the availability of the specified article on the chosen online marketplace at predefined intervals.
* **Email Notifications**: It sends email notifications to registered users when the article becomes available, providing a direct link to the product.
* **Customization**: Users can configure the program to monitor various articles on different marketplaces.

## Applications
* **E-commerce Enthusiasts**: Helps e-commerce shoppers track the availability of desired products and make purchases as soon as they become accessible.
* **Collectors and Hobbyists**: Enables collectors and hobbyists to stay up-to-date with new releases and limited-edition items on various marketplaces.
* **Price-Sensitive Shoppers**: Allows users to monitor articles until they reach a specific price threshold, ensuring they get the best deal.

##¬†How to Use
* Install the required dependencies. You can use pip to install them from the requirements.txt file.
```
pip install -r requirements.txt
```
* Configure the program by editing the **config.py** file. Add your email credentials, specify the article details, and choose the online marketplace to monitor.
* Run the program.
```
python main.py
```
* Sit back and relax! You'll receive email notifications as soon as the specified article becomes available on the chosen online marketplace.


### Development Details

**Technology Stack**
* **Programming Language**: Python
* **Email Sending**: SMTP for sending email notifications.
